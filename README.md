## 🧩 代表项目

### 🌟 RankMixer
PyTorch 实现的 RankMixer 排序模型，用于理解工业推荐场景下如何扩展 ranking model。  
核心包括 Multi-Head Token Mixing、Per-token FFN、Sparse-MoE 和 ReLU Routing。
